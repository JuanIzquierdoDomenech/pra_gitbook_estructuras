# Introducción

<figure><img src=".gitbook/assets/logos.png" alt=""><figcaption></figcaption></figure>

En la Práctica 3 trabajaremos los **conceptos y técnicas de estructuras de datos "Tablas Hash"** y **"Árboles Binarios de Búsqueda"** que hemos aprendido en el bloque 2 de teoría. La práctica se divide en dos partes:

* [**Parte 1: TAD Diccionario mediante Tablas Hash**](parte-1-tad-diccionario-con-t.-hash/).
* [**Parte 2: TAD Diccionario con Árboles Binarios de Búsqueda**](parte-2-tad-diccionario-mediante-abb/)**.**

Como pre-requisito, **deberéis haber realizado previamente la** [**Práctica 0**](https://app.gitbook.com/o/uY9F7U6WRW2jZBN5gWuV/s/SyyZos0HKvzFTjkdzLNv/), para familiarizaros con las herramientas del entorno de trabajo: sistema operativo Linux, terminal Bash, programación en C++, edición de ficheros de código fuente con Vim, control de versiones con Git, y automatización de procesos de compilación con Make.

También es imprescindible haber **finalizado la implementación de la clase `ListLinked<T>`** de la [**Práctica 1**](https://app.gitbook.com/o/uY9F7U6WRW2jZBN5gWuV/s/sfbBaRe5dlDS9ZlSiKn6/) para poder implementar la Tabla Hash propuesta en la parte 1 de esta práctica.&#x20;

Alternativamente, si por alguna razón esta no se ha implementado correctamente, puede utilizarse la clase [**`std::list<T>`** ](https://cplusplus.com/reference/list/list/)de la librería estándar de C++, que implementa una lista basada en nodos enlazados con otra interfa&#x7A;_._
