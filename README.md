# Introducción

<figure><img src=".gitbook/assets/logos.png" alt=""><figcaption></figcaption></figure>

En la Práctica 3 trabajaremos los **conceptos y técnicas de estructuras de datos "Tablas Hash"** y **"Árboles Binarios"** que hemos aprendido en el bloque 2 de teoría. La práctica se divide en dos partes:

* **Parte 1: TAD Diccionario mediante Tablas Hash**.
* **Parte 2**: **Árboles Binarios**.

Como prerrequisito, **deberéis haber realizado previamente la** [**Práctica 0 ("Entorno de trabajo")**](https://app.gitbook.com/o/X8vAd4LY1sPkw0QLbSUw/s/KCglCYAzIkqjjsArrX0I/), para familiarizaros con las herramientas del entorno de trabajo: sistema operativo Linux, terminal Bash, programación en C++, edición de ficheros de código fuente con Vim, control de versiones con Git, y automatización de procesos de compilación con Make.

También es imprescindible haber **finalizado la implementación de la clase** [**ListLinked\<T>**](https://app.gitbook.com/s/XJ8cratcuUw79RTeMnO7/parte-1-edl-lista-generica/clase-listlinked-less-than-t-greater-than) de la [**Práctica 1**](https://app.gitbook.com/o/X8vAd4LY1sPkw0QLbSUw/s/XJ8cratcuUw79RTeMnO7/) para poder implementar la Tabla Hash propuesta en la parte 1 de esta práctica. Alternativamente, si por alguna razón esta no se ha implementado correctamente, puede utilizarse la clase [**`std::list<T>`** ](https://cplusplus.com/reference/list/list/)de la librería estándar de C++, que implementa una lista basada en nodos enlazados con otra interfa&#x7A;_._
